Projects

Spectrum photo editor

Android application that converts regular photos to artistic images using Neural Style Transfer. The neural network runs completely on-device, leveraging the power of depthwise-separable convolutions. I used tensorflow-lite as the inference library and have also added GPU-support for supported mobile-devices.

Available on Amazon Appstores

Signature verification

Writer independent off-line Signature verification using Siamese Convolutional networks. The model was trained on the benchmark CEDAR1 dataset and achieved a AUC of 75.34%.

Cancer detection and Diagnosis - Mylan hackathon '18

Made for Mylan Hackathon 2018 organised by Mylan Pharmeceuticals. The project is a flask website for showcasing some of the machine learning models we built for early detection of cancer through lifestyle habits and non-invasive tests. We used Convolutional Neural networks for diagnosing cancer through patient's tissue samples. We also built a model for binary and multiclass classification of chemotheraphy drugs(protein kinases) using their drug fingerprints.

View Project

Context based Question-Answering system

Context based question answering system trained on the Stanford Question-Answering Dataset(SQUAD2.0). Used a GRU cell based bidirectional recurrent neural network with an Attention mechanism(dot product attention). The model was written in Keras, and preprocessing was done using Keras's Tokenizer API.

View Project

Image color quantization using K-means

Implemented the K-Means algorithm in C++ to convert a given RGB image into a K-bit RGB image, thus quantizing all colours in the image to the K most dominant colours(hence compressing the image file size in the process). Used OpenCV's Mat class for representing and processing images.

Denoising Autoencoder

A denoising autoencoder trained on the RENOIR dataset(Mi3 subset). The model was written entirely in Tensorflow and was trained on a GPU on Google colab.
